»Help?

Type [command] + Enter

  • 'home' -- Home page for the CLI
  • 'list_links' -- List your favorite links ex: 'list_links'
  • 'list_category' -- List your category ex: 'list_category'
  • 'add' -- Adding a new link to the favorite list ex: 'add <url> <category_id or name>'
  • 'update' -- Update a favorite link ex: 'update <url_id> <url> <category_id or name>'
  • 'remove' -- Remove a favorite link ex: 'remove <url_id>'
  • 'search' -- Search for a favorite link
  • 'filter' -- Filter your favorite links
  • 'help' -- displays this list
